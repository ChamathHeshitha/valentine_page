# Valentine's Day Website

To see the website:
1. Open the `index.html` file in your browser (Chrome, Safari, etc.).

## Customization
- **Photos**: Replace the image URL in `index.html` (line 25) with your own couple photo. You can put a file named `couple.jpg` in this folder and change the src to `src="couple.jpg"`.
- **Text**: You can edit the romantic message in `index.html`.
